# Historias de Usuario - Integración

### US-H-001: Recibir un correo electrónico cuando mi pago sea aprobado
- **Como** cliente
- **Quiero** recibir un correo electrónico cuando mi pago sea aprobado
- **Para** tener un comprobante escrito del estado de mi compra
- **Criterios de Aceptación**:
  - El correo debe incluir el número de pedido, el detalle de productos y el monto final pagado.
  - El formato del correo debe ser HTML responsivo legible en clientes móviles.

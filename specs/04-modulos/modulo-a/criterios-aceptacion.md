# Criterios de Aceptación - Catálogo

Este documento define las condiciones que el componente debe cumplir para considerarse finalizado.

## Criterios de Calidad y Rendimiento
- **CA-A-001**: La grilla del catálogo debe ser totalmente responsiva usando Bootstrap 5.3 (1 columna en móviles, 2 en tablets, 3 o 4 en escritorio).
- **CA-A-002**: La respuesta del endpoint API `/api/productos` debe tardar menos de 200ms en condiciones normales de red.
- **CA-A-003**: La navegación entre páginas del catálogo debe mantener los filtros previamente seleccionados.

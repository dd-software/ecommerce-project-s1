# Gestión de Riesgos - Autenticación

## Identificación y Mitigación de Riesgos
### Riesgo: Exposición del token JWT por robo de almacenamiento local.\n- **Impacto**: Alto\n- **Estrategia de Mitigación**: Configurar tiempos de expiración cortos y invalidar tokens del lado servidor si es necesario.\n
